# 🚀 21 Days Frontend Challenge

## 📌 About
This repository contains all projects built during my **21 Days Frontend Challenge** where I'm mastering **HTML, CSS, and JavaScript** through daily real-world projects.

## 🎯 Challenge Goals
- ✅ Build 21 real-world projects in 21 days
- ✅ Learn industry-level flow planning (Flow → Build)
- ✅ Master JavaScript with practical implementation
- ✅ Create portfolio-ready projects
- ✅ Daily consistency & documentation

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage API
- REST APIs (later days)

## 📂 Project Structure
Each day's folder contains:
- `index.html` - Structure
- `style.css` - Styling
- `script.js` - Logic
- `flow.md` - Planning document
- `screenshot.png` - Preview

## 💡 Key Learnings
- Planning before coding saves time
- Edge cases handling is crucial
- localStorage for data persistence
- Clean code practices

## 🚀 How to Run
1. Clone this repo
2. Navigate to any day's folder
3. Open `index.html` in browser
